# ITAS API

ITAS prototype API

## Installation

1. Clone this repository: `git clone https://github.com/sir3mat/ITAS-API.git`
2. Install dependencies: `pip install -r requirements.txt`

## How to use
```
python manage.py runserver
```

### API Endpoints
- `/api/`
- `/api/pathFinder`
- `/api/map`
- `/api/roads`
- `/api/onlinePathFinder`
- `/api/isConnected`
- `/api/spec/`
- `/api/spec/swagger-ui`
